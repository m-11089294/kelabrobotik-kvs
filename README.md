SISTEM PENDAFTARAN PELAJAR KE KELAB ROBOTIK KOLEJ VOKASIONAL SEPANG

Cara menggunakan sistem ini:

1. Masuk sistem index.html
2. Log masuk dan lengkapkan borang yang disediakan di register.html
3. tekan butang hantar untuk menghantar borang
4. tekan butang reset untuk memadam semua maklumat dalam borang
5. senarai pelajar yang telah menghantar borang akan terpaparkan di members.html
